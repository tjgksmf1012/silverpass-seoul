실버패스 서울 Care 보호자·돌봄기관과 함께 쓰는 고령자 생활권 이동 AI 코파일럿 2026 서울시 빅데이터 활용 경진대회 | 창업부문 제출용 초안 핵심 문제 고령자 이동은 “가장 빠른 길”보다 저상버스, 승강기, 날씨, 경사, 생활 편의시설 접근성이 더 중요하지만 기존 지도 서비스는 이를 충분히 반영하지 못한다. 핵심 해결책 실시간 교통약자 정보와 생활권 데이터를 결합해 “가장 덜 힘든 이동안”을 추천하고 보호자·기관과 공유한다. 초기 고객 주야간보호센터·방문요양 서비스 업체 수익 구조 기관 구독형 SaaS + 안내 위젯/링크 서비스 + 대시보드 확장 1. Executive Summary 실버패스 서울 Care는 고령자·보호자·돌봄기관이 함께 쓰는 생활권 이동 코파일럿이다. 서울 열린데이터광장의 고령자 통계, 노인여가복지시설, 버스도착정보, 교통약자용 도착예정정보, 승강기 가동현황, 실시간 도시데이터, 공중화장실 위치정보와 공공데이터포털의 약국 정보를 결합해, 사용자의 반복 목적지 이동을 더 안전하고 덜 힘들게 만든다. 고령자 관점에서 중요한 것은 “가장 빠른 길”이 아니라 “가장 덜 힘든 길”이다. 실시간 저상버스/승강기 상태, 날씨, 경사, 개인 이동 제약을 동시에 반영한다. 음성 입력에만 의존하지 않고 큰 버튼 퀵메뉴·즐겨찾기 카드·보호자 대리설정으로 STT 한계를 보완한다. 응급 상황에서는 AI가 판단하지 않고 119 연결 및 보호자 호출로 즉시 전환하는 안전 가드레일을 적용한다. 초기 고객은 주야간보호센터·방문요양업체로 설정해 도입 속도와 매출 현실성을 높인다. 2. 문제 정의 고령자 이동의 실패는 일반적인 길찾기 실패와 성격이 다르다. 계단 하나, 저상버스 대기시간, 승강기 고장, 폭염, 공중화장실 부재가 모두 이동 포기의 원인이 된다. 특히 보호자와 돌봄기관은 반복 이동 안내를 매번 전화와 문자로 처리하고 있어 업무 부담이 크다. 핵심 기능 설명 반복 이동 템플릿 복지관·병원·단골 약국·집 등 자주 가는 목적지를 큰 버튼으로 배치해 음성 실패 시에도 즉시 사용 가능 실시간 이동 추천 정류소별 교통약자용 도착예정정보와 승강기 가동현황을 반영해 현재 시점에서 가장 덜 힘든 이동안 제시 개인화 프로필 최대 도보시간, 계단 허용 수준, 보행보조기구 사용 여부, 보호자 공유 여부를 기억해 맞춤형 추천 응급 가드레일 응급 키워드 인식 시 길안내를 중단하고 119 연결 및 보호자 연락으로 즉시 전환 현장 피드백 루프 입구 계단, 공사, 폐쇄 등 공공데이터의 공백을 보호자·기관·사용자 피드백으로 보완 3. 왜 지금 필요한가 서울시는 고령자 현황을 정기 통계로 관리하고 있으며, 노인여가복지시설, 저상버스, 실시간 버스·승강기 정보 등 이미 서비스에 바로 쓸 수 있는 데이터 기반을 갖추고 있다. 따라서 이 주제는 단순 복지 아이디어가 아니라, 실제 도시 데이터 위에 구축 가능한 생활지원 서비스다. 직접 사용자 고령자 본인 실제 구매/도입자 보호자, 주야간보호센터, 방문요양업체 2차 확장 고객 복지관, 실버타운, 요양기관, 자치구 도입 이유 반복 안내 업무 감소, 이동 실패 감소, 보호자 안심, 기관 운영 효율 향상 4. 데이터 활용 및 융합 설계 본 서비스는 인구·복지, 교통, 환경, 생활·보건 데이터를 결합하는 구조로 설계되어 공모전의 이종 데이터 결합 가점 요건에도 부합한다. 정적 데이터로 기본 후보군을 만들고, 실시간 데이터로 “지금 실제로 가능한 이동안”을 재정렬한다. 데이터 분야 주요 데이터셋 활용 목적 인구·복지 서울시 고령자현황 통계 / 독거노인 현황 / 노인여가복지시설 수요 지역, 타깃 사용자, 반복 이동 목적지 도출 교통 저상버스 도입 노선 및 보유율 / 버스도착정보 / 교통약자용 도착예정정보 / 승강기 가동현황 지금 실제로 탈 수 있는 이동수단과 대기시간 계산 환경 서울 실시간 도시데이터(날씨) / 무더위쉼터 / 경사도 폭염·강우·언덕길 부담을 반영한 이동 부담도 산출 생활·보건 서울시 공중화장실 위치정보 / 약국 정보 / 장애인 편의시설 데이터 라스트마일 접근성과 생활 편의성 반영 5. AI 및 서비스 동작 방식 질의 해석: “오늘은 덜 걷는 길로 약국 가고 싶어”와 같은 자연어를 목적지·조건·긴급도로 구조화 개인화 추천: 최대 도보시간, 계단 허용 수준, 보행보조기구 사용 여부, 반복 목적지 기록을 반영해 경로/목적지를 재정렬 설명 생성: 추천 이유를 쉬운 문장으로 제공해 고령자·보호자 모두가 이해하기 쉽게 만듦 안전 가드레일: 응급 키워드 인식 시 일반 안내를 중단하고 119 연결 및 보호자 연락으로 전환 피드백 루프: 공공데이터가 포착하지 못한 현장 변수(입구 계단, 공사, 폐쇄)를 기관·보호자·사용자 피드백으로 보완 서비스 플로우는 “프로필 설정 → 목적지 선택 → 이동 부담도 계산 → 추천 설명 → 보호자 공유/기관 기록” 순으로 단순화한다. 6. UX 및 차별성 실버패스 서울 Care의 차별점은 지도 경로 엔진 자체보다 “반복 이동을 관리하는 개인화 기억력”과 “돌봄 워크플로우”에 있다. 네이버/카카오가 교통약자 모드를 만들 수 있어도, 보호자와 기관이 같은 프로필을 함께 관리하고 반복 목적지를 공유하는 구조는 별도의 가치가 있다. 하이브리드 UI: 음성 입력 + 대형 버튼 + 사진형 즐겨찾기 카드 + 최근 경로 재사용 개인 프로필 기억: 도보·계단·환승 허용 기준을 누적 저장 기관 관점 운영성: 어르신별 이동 템플릿과 보호자 안심 리포트 제공 Zero-touch 온보딩: 설치 없는 링크형 웹앱/카카오톡 진입, 보호자 대리설정, 짧은 대화형 튜토리얼 7. 비즈니스 모델 및 시장 진입 전략 초기부터 B2G를 주요 매출원으로 잡지 않고, 도입 의사결정이 빠르고 Pain Point가 선명한 주야간보호센터·방문요양업체를 1차 고객으로 설정한다. 이후 운영성과 레퍼런스를 확보한 뒤 복지기관과 공공기관으로 확장한다. 단계 고객 가치 제안 1차 타깃 주야간보호센터·방문요양업체 어르신 이동 프로필 관리, 보호자 안심 리포트, 링크형 반복 동선 안내 2차 확장 복지관·실버타운·요양기관 입소자·방문자 이동 지원, 문의 감소, 기관용 안내 위젯 3차 확장 자치구·공공기관 이동 사각지대 대시보드, 정책 예산 편성 근거, 민원 유형 분석 8. MVP 범위 및 구현 계획 사용자 프로필 등록: 최대 도보시간, 계단 허용, 보행보조기구, 자주 가는 곳 3개 반복 목적지 퀵 실행: 복지관·병원·약국·집 버튼 실시간 추천 엔진: 교통약자용 도착예정정보, 승강기 가동현황, 날씨, 경사 반영 보호자 공유 화면: 목적지, 예상 소요시간, 권장 이동안, 도착 상태 응급 전환 화면: 응급 키워드 감지 시 119/보호자 호출 1단계 링크형 웹앱 MVP 구축 및 보호자·기관 테스트 2단계 실시간 데이터 연동 고도화 및 현장 피드백 루프 적용 3단계 기관 대시보드·사각지대 시각화·정책 연계 기능 확장 9. 리스크와 대응 방안 STT 오인식 위험 → 하이브리드 UI, 큰 버튼, 즐겨찾기 카드, 보호자 대리설정으로 보완 공공데이터 라스트마일 한계 → 기관 검증 + 사용자 피드백 + 접근성 보조 데이터 병행 응급 안내의 책임 문제 → 응급 판단을 수행하지 않고 119 연결로 즉시 전환하는 안전 가드레일 적용 현장 도입 마찰 → 설치 없는 링크형 온보딩과 최소 입력 구조로 복지사의 업무를 늘리지 않도록 설계 10. 기대효과 고령자: 이동 포기 감소, 대기 불안 감소, 반복 이동 성공률 향상 보호자: 반복 설명 감소, 안심감 확보, 비응급 상황 대응 용이 돌봄기관: 등하원·내원 동선 관리 효율 향상, 문의 감소, 운영 데이터 축적 공공부문: 승강기·경사로·쉼터·안내 체계 개선이 필요한 사각지대 파악 가능 실버패스 서울 Care는 단순 길찾기 앱이 아니라, 서울시의 공공데이터를 생활복지 인프라로 전환하는 서비스다. 11. 참고 데이터 및 공식 페이지 2026 서울시 빅데이터 활용 경진대회 안내: https://news.seoul.go.kr/gov/archives/576001 서울시 고령자현황 통계: https://data.seoul.go.kr/dataList/10730/S/2/datasetView.do 서울시 버스도착정보 조회서비스(API): https://data.seoul.go.kr/dataList/OA-1091/L/1/datasetView.do 서울교통공사 교통약자 이용시설 승강기 가동현황: https://data.seoul.go.kr/dataList/OA-15994/S/1/datasetView.do 서울시 공중화장실 위치정보: https://data.seoul.go.kr/dataList/OA-22586/S/1/datasetView.do 서울 실시간 도시데이터: https://data.seoul.go.kr/dataList/OA-21285/F/1/datasetView.do 전국 약국 정보 조회 서비스(공공데이터포털): https://www.data.go.kr/data/15000576/openapi.do 전국 응급의료기관 정보 조회 서비스(공공데이터포털): https://www.data.go.kr/data/15000563/openapi.do
