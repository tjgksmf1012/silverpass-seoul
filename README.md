다운로드 하는법(순서대로)

1.코드 받기
git clone https://github.com/tjgksmf1012/silverpass-seoul.git
cd silverpass-seoul

2. 패키지 설치
npm install

3. .env파일 만들기
VITE_ANTHROPIC_API_KEY=
VITE_SEOUL_API_KEY=
VITE_GONGGONG_API_KEY=

4. 실행
npm run dev

[작업 후 저장하는 법]
git add .
git commit -m "작업 내용"
git push

[다른 컴퓨터에서 최신 코드 받는 법]
git pull
